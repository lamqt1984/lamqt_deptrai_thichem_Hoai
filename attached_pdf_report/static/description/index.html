Automatically attach PDF file when you download PDF Reports on odoo system

All pdf reports on odoo use

+ Qweb report

+ Jasper eport

+ Webkit report

Main Features by example

Step 01: User click download report pdf

Step 02: System auto attach information of this report. User can go menu "Auto Attached PDF Report" to check.

Step 03: You can not find the old reports, you can go menu "Auto Attached PDF Report" to download again
